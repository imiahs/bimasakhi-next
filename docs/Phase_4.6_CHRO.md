# 🔷 4.6 Chief Human Resources Officer (CHRO)

## Objective

Acquire, develop, manage, and retain high-performing human resources aligned with business systems.

## System Nature

* 60–70% Rule-based execution
* 20–30% Automation (HRMS + workflows)
* 10–20% Human judgment

## Core Output

Hiring → Training → Performance → Retention

## Core Dependencies

* CSO → Sales hiring demand
* COO → Operational manpower
* CFO → Salary & cost control
* CTO → HR systems
